**HTML and CSS Roadmap**

This is a high-intensity, "no-fluff" 2-Week Professional HTML \& CSS Roadmap. I

have stripped away deprecated tags and useless properties to focus strictly on

what we use in modern software companies (Flexbox, Grid, Semantic HTML, and

Responsive Design).



**Week 1: The Structural Foundation \& Layout Mastery**



**Day 1: Semantic HTML5 \& Document Structure - Done**



Focus: Writing code that Google and Screen Readers understand.



&#x20; - Study: Document structure, <header>, <nav>, <main>, <section>, <article>,

&#x20;   <footer>, and SEO meta tags.

&#x20; - Exercise 1 (Easy): Create a basic page structure using only semantic tags

&#x20;   (no <div> yet).

&#x20; - Exercise 2 (Medium): Build a blog post structure with a title, author

&#x20;   metadata, body text, and a sidebar.

&#x20; - Exercise 3 (Hard): Create an accessible FAQ list using <details> and

&#x20;   <summary> tags.

&#x20; - Project: Personal Resume Structure. Build a full resume using only HTML (no

&#x20;   CSS). It must be perfectly structured for a screen reader.

\--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------



**Day 2: CSS Fundamentals \& The Box Model**



Focus: Understanding how elements occupy space.



&#x20; - Study: Selectors (.class, #id, \*), Colors, Units (px, rem, %), and the Box

&#x20;   Model (Margin, Border, Padding, Content).

&#x20; - Exercise 1: Create 3 boxes and give them different margins and paddings to

&#x20;   see how they push each other.

&#x20; - Exercise 2: Use box-sizing: border-box and explain why it’s used in every

&#x20;   professional project.

&#x20; - Exercise 3: Style a button with hover states using :hover and :active.

&#x20; - Project: Styled Profile Card. Create a card with an image, name, and

&#x20;   "Follow" button using specific padding/margins.

\--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------



**Day 3: Typography \& Google Fonts**



Focus: Making text look professional.



&#x20; - Study: Font-family, font-weight, line-height, letter-spacing, and importing

&#x20;   Google Fonts.

&#x20; - Exercise 1: Import "Inter" or "Roboto" from Google Fonts and apply it to a

&#x20;   page.

&#x20; - Exercise 2: Create a hierarchy: h1 (large/bold), h2 (medium), p (readable

&#x20;   line-height).

&#x20; - Exercise 3: Style a blockquote with a unique left border and italicized

&#x20;   text.

&#x20; - Project: Article Landing Page. Recreate a Medium.com style article layout

&#x20;   with perfect typography and spacing.

\--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------



**Day 4: Flexbox Mastery (Part 1: The Basics)**



Focus: The most important tool in modern CSS.



&#x20; - Study: display: flex, justify-content, align-items, flex-direction.

&#x20; - Exercise 1: Center a div perfectly in the middle of the screen (The "Holy

&#x20;   Grail" of CSS).

&#x20; - Exercise 2: Create a Navigation Bar with a logo on the left and links on the

&#x20;   right using justify-content: space-between.

&#x20; - Exercise 3: Build a vertical sidebar menu using flex-direction: column.

&#x20; - Project: Social Media Header. Build a functional header with search bar,

&#x20;   logo, and profile icons using Flexbox.

\--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------



**Day 5: Flexbox Mastery (Part 2: Complex Layouts)**



Focus: Building real UI components.



&#x20; - Study: flex-wrap, flex-grow, flex-shrink, gap.

&#x20; - Exercise 1: Create a gallery of 6 images that wraps to the next line when

&#x20;   space runs out.

&#x20; - Exercise 2: Create a "Pricing Table" where the middle card grows larger than

&#x20;   the others using flex-grow.

&#x20; - Exercise 3: Build a comment section with a profile picture on the left and

&#x20;   text on the right.

&#x20; - Project: Features Section. Build a 3-column feature section for a website

&#x20;   (Icon + Title + Text).

\--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

**Day 6: CSS Grid (The Powerhouse)**



Focus: Controlling 2D layouts (Rows and Columns).



&#x20; - Study: grid-template-columns, grid-template-rows, repeat(), fr units,

&#x20;   grid-gap.

&#x20; - Exercise 1: Create a 3x3 square grid.

&#x20; - Exercise 2: Create a "Holy Grail Layout" (Header, Sidebar, Main Content,

&#x20;   Footer) using grid-template-areas.

&#x20; - Exercise 3: Build a Masonry-style image gallery.

&#x20; - Project: Dashboard Layout. Build a professional admin dashboard layout with

&#x20;   a sidebar, top bar, and 4 main data widgets.

\--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------



**Day 7: Week 1 Review \& Mini-Site**



&#x20; - Study: Review everything. Clean up code naming using BEM (Block Element

&#x20;   Modifier) basics.

&#x20; - Project: Small Business Landing Page. Combine Flexbox and Grid to build a

&#x20;   one-page site for a "Coffee Shop" including a hero section, menu (Grid), and

&#x20;   footer.



\--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------



**Week 2: Responsiveness \& Professional Polish**



**Day 8: Responsive Design (Media Queries)**



Focus: Making sites work on iPhone, iPad, and Desktop.



&#x20; - Study: Viewport meta tag, @media rules, Mobile-first workflow.

&#x20; - Exercise 1: Change the background color of a page based on screen width.

&#x20; - Exercise 2: Hide a sidebar on mobile and show it on desktop.

&#x20; - Exercise 3: Convert a 3-column grid into a 1-column list for mobile devices.

&#x20; - Project: Responsive Portfolio Gallery. A gallery that shows 1 image per row

&#x20;   on mobile, 2 on tablet, and 4 on desktop.



\--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------



**Day 9: Forms \& User Input**



Focus: Creating functional data entry points.



&#x20; - Study: <input>, <textarea>, <select>, <label>, and styling inputs (focus

&#x20;   states, outlines).

&#x20; - Exercise 1: Build a login form with "Email" and "Password" and a "Forgot

&#x20;   Password" link.

&#x20; - Exercise 2: Create a custom checkbox and radio button style (Industry

&#x20;   standard skill).

&#x20; - Exercise 3: Add validation styling (e.g., red border if the input is

&#x20;   :invalid).

&#x20; - Project: Modern Contact Page. Build a contact form with a Map (placeholder)

&#x20;   on one side and a styled form on the other.



\--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------



**Day 10: CSS Variables \& Transitions**



Focus: Scalability and smooth UX.



&#x20; - Study: :root variables, transition: all 0.3s ease, and transform.

&#x20; - Exercise 1: Create a "Dark Mode" toggle using only CSS variables.

&#x20; - Exercise 2: Create a button that grows slightly and changes color smoothly

&#x20;   when hovered.

&#x20; - Exercise 3: Build a "Card" that flips or lifts up when hovered.

&#x20; - Project: Interactive Product Card. Build an E-commerce card where the image

&#x20;   zooms slightly and the "Add to Cart" button slides up on hover.

\--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------



**Day 11: Real-World Workflow \& BEM**



Focus: Writing clean, maintainable code.



&#x20; - Study: BEM Naming Convention (header\_\_btn--large), CSS resets

&#x20;   (Normalize.css).

&#x20; - Exercise 1: Take a messy piece of HTML/CSS and rename everything using BEM.

&#x20; - Exercise 2: Organize a CSS file into sections (Variables, Base, Layout,

&#x20;   Components).

&#x20; - Exercise 3: Use a CSS Reset to ensure your site looks the same in Chrome and

&#x20; - Project: Refactor Day 7 Project. Take your Coffee Shop site and rewrite the

**Day 12: Modern Components (Modals \& Tooltips)**

3 Golden Rules for this Roadmap:



1\.  Don't use float: It's outdated for layouts. Stick to Flexbox and Grid.

2\.  Inspect Element: Spend 30 minutes every day in Chrome DevTools looking at

&#x20;   how professional sites (like Airbnb or Stripe) are built.

3\.  No Copy-Paste: Type every line of code. Muscle memory is how you become a

&#x20;   professional.



After these 14 days, you will be ready to move on to JavaScript. Good luck!
